# DoHead (v1.2) – Editor

> Editor rápido y automático para generar rápidamente **cabezas "arregladas" y custom de Minecraft**.

---

## ¿Qué hace?

* Genera cabezas slim "bug" totalmente funcionales al instante
* Vista previa en tiempo real
* Convertidor en Base64 implementado en el código (codificación y decodificación)

---

## Lenguajes de programación usados

* HTML, CSS y JS

---

## Ejecutar

```bash
index.html (localmente)
https://glsk-dev.github.io/DoHead/ (online)
```
